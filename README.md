# 👁 Show Password Extension

Простое и лёгкое **расширение для браузера Chrome**, которое позволяет **показать пароль вместо точек или звёздочек** на любой странице.

---

## ❓ Если вы когда-нибудь задавались вопросом

* как **увидеть пароль вместо точек**
* как **показать пароль в поле password**
* как **отобразить скрытый пароль на сайте**
* как **сделать show password в браузере**
* как **раскрыть пароль в форме входа**

— это расширение решает эту задачу **одним кликом**.

---

## 🔎 Как это работает

Пароли в браузере обычно скрываются символами:

```
••••••
```

или

```
****
```

Это происходит потому что поле имеет тип:

```html
<input type="password">
```

Расширение временно меняет его на:

```html
<input type="text">
```

После этого вместо точек отображается **реальный текст пароля**.

---

## ⚡ Возможности

* 👁 **Показать пароль вместо точек**
* ⚡ **Один клик по иконке расширения**
* 🔑 Работает на **любых сайтах и формах входа**
* 🔄 Повторный клик **скрывает пароль обратно**
* 🪶 **Минимальный размер и быстрый запуск**
* 🔒 **Не собирает и не отправляет данные**

Все операции выполняются **локально в браузере**.

---

## 🧰 Когда это полезно

Расширение может пригодиться если нужно:

* проверить **введённый пароль**
* **увидеть пароль вместо звёздочек**
* проверить **автозаполнение браузера**
* **отладить форму логина**
* **вспомнить введённый пароль**

---

## 🔒 Безопасность

Расширение:

* ❌ не хранит пароли
* ❌ не отправляет данные на сервер
* ❌ не имеет сторонних зависимостей

Все изменения происходят **только в текущей вкладке браузера**.

---

## 📦 Установка

1. Скачать репозиторий

```
git clone https://github.com/yourusername/show-password-extension
```

2. Открыть страницу расширений Chrome

```
chrome://extensions
```

3. Включить **Developer mode**

4. Нажать **Load unpacked** и выбрать папку проекта.
